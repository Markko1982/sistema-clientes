# Aliases do projeto sistema-clientes

np          -> entra na pasta do projeto
c1          -> abre o projeto no VS Code
bs          -> roda buscar_por_sobrenome.py (busca de clientes por sobrenome)
menucli     -> abre o menu interativo (menu.py)
vproj       -> ativa o venv (~/.venvs/novo-projeto) e entra na pasta do projeto
aliasescli  -> mostra este arquivo ALIASES.md

Como usar (exemplos):

  np
  c1
  bs
  menucli
  vproj
  aliasescli
